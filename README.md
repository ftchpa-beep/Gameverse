# Gameverse

## Project Description
Gameverse is a multiplayer Roblox-style game that offers players a rich and interactive gaming experience in a vast virtual world.

## Features
- **Multiplayer Gameplay**: Play with friends and people from around the world.
- **Customizable Avatars**: Personalize your character with various outfits and accessories.
- **Diverse Environments**: Explore different worlds filled with unique themes and gameplay mechanics.
- **Engaging Quests**: Complete missions and quests to earn rewards and enhance your gameplay.
- **User-Generated Content**: Create and share your own levels and experiences with the community.

## Setup Instructions
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/ftchpa-beep/Gameverse.git
   ```
2. **Install Dependencies**:
   Navigate to the project directory and install required packages using:
   ```bash
   npm install
   ```
3. **Run the Game**:
   Use the following command to start the game:
   ```bash
   npm start
   ```

## Gameplay Guide
- Use the **WASD** keys or arrow keys to move your character.
- Interact with objects by pressing the **E** key.
- Communicate with other players using the in-game chat feature.
- Participate in events and challenges to earn in-game rewards.

Enjoy your adventure in Gameverse!
